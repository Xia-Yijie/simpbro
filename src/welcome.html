simpbro — 终端里的浏览器

一个用 Rust 写的 TUI 浏览器，支持 HTML 解析、CSS 排版、JavaScript 执行和 表单交互。

快捷键

  • g — 输入网址
  • Tab / Shift+Tab — 切换焦点（链接 / 输入框 / 按钮）
  • Enter — 激活焦点（跟随链接 / 编辑输入框 / 点击按钮）
  • j k 或 ↑ ↓ — 滚动
  • d u — 快速滚动（10 行）
  • b — 后退
  • s — 提交表单
  • q — 退出

试一下

  • example.com — 最简 HTML 页面
  • Hacker News — 经典链接列表
  • 百度 — 表单 + 搜索演示
  • GitHub — 复杂 JS 页面

关于

鼠标支持：滚轮滚动、单击聚焦、双击激活、拖拽选中后右键复制。
