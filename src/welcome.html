simpbro — 终端里的浏览器

一个用 Rust 写的 TUI 浏览器，支持 HTML 解析、CSS 排版、JavaScript 执行和 表单交互。

键盘操作

  • Tab / Shift+Tab — 切换焦点（链接 / 输入框 / 按钮）
  • Enter — 激活焦点（跟随链接 / 编辑输入框 / 点击按钮）
  • Ctrl+G — 切换栏目焦点（操作栏 ⇌ 地址栏）
  • ↑ ↓ — 滚动
  • Ctrl+Z — 后退
  • Ctrl+T — 新标签页
  • Ctrl+W — 关闭当前标签页
  • Ctrl+Tab / Ctrl+Shift+Tab — 切换标签页
  • Ctrl+C — 退出

鼠标操作

  • 滚轮 — 滚动内容
  • 单击链接 / 输入框 / 按钮 — 聚焦
  • 单击操作栏 / 地址栏 / 内容 — 切换栏目焦点
  • 点击 [回退] 或 [刷新] 按钮 — 后退 / 刷新当前页
  • 双击 — 激活焦点
  • 拖拽 — 选中文字
  • 选中后右键 — 复制到剪贴板
